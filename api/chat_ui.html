Foldseek Agent 工作台

左侧是 LLM 驱动的自然语言控制台，可以直接描述要执行的 Foldseek 操作。右侧是显式模块执行面板，适合精确控制参数。上方的文件上传区会把文件落到服务器工作目录，并返回可直接复用的服务器路径。

文件上传

用于上传查询结构、聚类输入文件或其他需要落到服务器上的本地文件。上传后会返回服务器绝对路径，你可以一键插入聊天框或模块 JSON。

检索数据库

检索类模块会优先使用这里的数据库选择。可以选配置好的数据库别名，也可以直接填写服务器上的 Foldseek 数据库前缀路径。原始 `.pdb.gz` / `.cif.gz` 文件目录本身不是 Foldseek 数据库；如果你手头只有这类文件，先运行 `createdb`，再对生成结果运行 `createindex`。

结构挖掘启动器

把搜索类型、查询结构、数据库、topk、临时目录和结果输出先确认好，再一键执行。执行后聊天区会自动拿到结果上下文，方便继续追问“为什么第一名更好”。当前启动器直接支持服务器上的结构文件路径或上方上传得到的路径。
